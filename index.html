⚠ Persediaan Guru (Setup)

Untuk sistem ini berfungsi, anda perlu menyediakan Google Apps Script sebagai backend API. Lihat fail apps-script.js yang disertakan.

  1. Buka Google Sheet anda → Extensions → Apps Script
  2. Tampal kod dari apps-script.js
  3. Deploy sebagai Web App (Execute as: Me, Access: Anyone)
  4. Salin URL web app dan ganti APPS_SCRIPT_URL di bawah
